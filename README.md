# Building Communities into Economies

**A Systematic Approach to Growth and Innovation**

> *"Sustainable growth and innovation don't come from top-down systems — they come from enabling communities to learn, share, and create value together."*

---

## About This Book

This is the official repository for *Building Communities into Economies*, a free, open-access book by [Growth Clinic](https://growth-clinic.github.io) documenting the **Growth and Innovation System (GIS)** — a framework for transforming communities into sustainable economies through coordinated learning, shared knowledge, and community-driven innovation.

The book draws on real experiences across Nigerian communities, tech ecosystems, and learning programmes to present a practical, repeatable system for community-led growth.

**Read the book online:** [growth-clinic.github.io/gis-book](https://growth-clinic.github.io/gis-book)

---

## What is the GIS?

The Growth and Innovation System has three interlocking components:

| Component | What it does |
|---|---|
| **Smart Micro-content** | Bite-sized, actionable tasks derived from real-world problems |
| **Local Contributors** | Community members, mentors, and partners providing context and knowledge |
| **Thinking Frameworks** | Mental models (Design, Market, User, Business, Project Thinking) that scaffold problem-solving |

Together, these components enable communities to move from shared pain points to homegrown innovative products and services.
